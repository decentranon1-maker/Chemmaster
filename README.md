# Chemistry Master

A comprehensive offline mobile learning application for chemistry education.

## Overview

Chemistry Master is a complete chemistry textbook and interactive learning platform that works entirely offline. It combines full textbook content with interactive tools, practice quizzes, and flashcards across multiple chemistry disciplines.

## Framework

Vite + React 19

## Features

### Complete Chemistry Library
- **Inorganic Chemistry**: Atomic structure, periodic trends, bonding, coordination compounds, transition metals, acids/bases, solid state, and descriptive chemistry
- **Organic Chemistry**: Nomenclature, stereochemistry, hydrocarbons, aromatics, reaction mechanisms, substitution/elimination, alcohols, ethers, carbonyl chemistry, carboxylic acids, and polymers
- **Physical Chemistry**: Gas laws, thermodynamics, kinetics, equilibrium, electrochemistry, phase diagrams, quantum chemistry, and surface chemistry

### Chapter Structure
Each chapter includes:
- Introduction and overview
- Step-by-step theory (basic to advanced)
- Diagrams and visualizations
- Worked calculations and examples
- Interactive tools and calculators
- Practice questions with explanations
- Quick revision summary
- Flashcards for memorization
- Real-world applications

### Interactive Tools
- Mole calculator
- pH calculator
- Ideal gas law calculator
- Energy transition calculator
- Orbital visualizers
- Unit converters

### Learning Features
- Offline-first: No internet connection required
- Dark/light mode support
- Adjustable font size
- Progress tracking
- Bookmarks and notes
- Practice quizzes with instant feedback
- Flashcard study system
- Search functionality

### User Interface
- Clean black and white academic theme
- Soft white backgrounds with charcoal text (light mode)
- Black backgrounds with white text (dark mode)
- Minimal accent highlights in cool grey and light blue
- Smooth transitions and rounded cards
- Generous spacing for comfortable reading
- Mobile-responsive design

## Theme

Academic black and white color scheme optimized for long study sessions. The interface uses neutral tones with subtle blue accents, providing a distraction-free learning environment suitable for extended reading periods.

## Data Storage

All user data is stored locally using browser localStorage:
- Reading progress
- Bookmarks
- Notes and highlights
- Settings preferences
- Quiz scores

No backend, cloud services, or user accounts required.
